# Эндпоинты задач

## Получить список задач
GET /tasks
**Параметры:**
- status (статус задачи)
- assignee_id (ID исполнителя)
- limit (количество)

## Создать задачу
POST /tasks
## Получить одну задачу
GET /tasks/{id}
## Обновить задачу
PUT /tasks/{id}
## Удалить задачу
DELETE /tasks/{id}
# **Тело запроса:**
{
  "title": "Название",
  "assignee_id": 123
}
