# Обзор API

## Основная информация
API для работы с системой управления задачами TaskTracker.

### Базовый URL
https://api.tasktracker.example.com/v1

### Аутентификация
Используйте API ключ в заголовке запроса:
Authorization: Bearer ваш_ключ

### Формат данных
Все данные в формате JSON.

### Коды ответов
- 200: Успешно
- 201: Создано
- 400: Ошибка запроса
- 401: Нет доступа
- 404: Не найдено
- 500: Ошибка сервера
