# Быстрый старт

Добро пожаловать в TaskTracker! Это руководство поможет вам создать первую задачу за 5 минут.

## 1. Создание учетной записи

Перейдите на [сайт приложения](https://app.tasktracker.example.com) и нажмите **"Зарегистрироваться"**.

![](../images/signup.png)

## 2. Создание первой задачи

После входа в систему:
1.  Нажмите кнопку **"Новая задача"** в верхней панели.
2.  Заполните поля:
    *   **Название:** "Проанализировать требования"
    *   **Описание:** "Собрать и задокументировать требования к новому модулю"
    *   **Исполнитель:** Выберите себя из списка.
3.  Нажмите **"Создать"**.
