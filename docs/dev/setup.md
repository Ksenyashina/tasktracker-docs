# Настройка среды разработки

В этом руководстве описаны шаги по настройке локальной среды разработки для проекта TaskTracker.

## Предварительные требования

Перед началом убедитесь, что на вашем компьютере установлено:

- **Git** (версия 2.30 или выше)
- **Python** (версия 3.8 или выше)
- **Docker Desktop** (опционально, для работы с контейнерами)
- **Редактор кода** (рекомендуется VS Code или PyCharm)

### Проверка установки

```bash
# Проверьте версию Git
git --version

# Проверьте версию Python
python --version
# или
python3 --version

# Проверьте установку pip
pip --version
